# blob_cheaters

Playing Primer's blob coin flipping game. The code runs the optimal (i think) strategy for the game using a recursive expectation table and has an EV of -0.128. It achieves a score of about 800 on average with some outliers as high as 100,000!

[Link](https://www.youtube.com/watch?v=QC91Bf8hQVo) to the video explaining the game and a sample of 1000 runs.
